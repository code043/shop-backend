# Backend Shop

API REST para gerenciamento de produtos, carrinho de compras e pedidos de uma loja virtual de camisas.

## Getting Started

### Requirements

- Vscode
- Node
- Nest
- Prisma

### Installation

```bash
git clone https://github.com/code043/creepy-backend.git
cd creepy-backend
npm install
npm run dev
```
